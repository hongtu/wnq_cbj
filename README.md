#储板机		
##0 背景		
老式的上板机需要人工往静电箱里面预装印刷好的板子，每台机器至少需要两个静电箱循环使用，并耗用较多时间预装		
##1 目标		
1、1 代替千层架储存板子		
1、2 节省人工		
##2 预期功能		
###2、1 流程		
2、1、1 印刷人员将印好的板子放置到机器左边的平台上		
2、1、2 机器将平台上的板子推进多层架中暂存		
2、1、3 在右边的机器缺料时，机器将暂存的板子按照进入的时间顺序将板子推出			
###2、2 安全		
2、2、1 设有上下极限。在触发任一极限后触发一多触点继电器，其中一路接步进电机驱动箱的停止端；有一路接单片机的检测端		
2、2、2 储板箱左右侧设置光电对管，对于超出的板，停止并报警		
###2、3 细节 		
2、3、1 步进电机的加速和减速		
2、3、2 上电后原点的寻找		
2、3、3 进板和出板逻辑		
2、3、4 进板和出板动作逻辑   
###2、4 程序流程   
>设置输入端口     
>调用初始化函数 并判断初始化函数返回值  
>调用进板和出板逻辑函数  
>调用上下动作函数  
>调用进板或出板动作函数
